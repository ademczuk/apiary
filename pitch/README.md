# Apiary Pitch Package

Pre-event pitch materials for Zero-One Hack Vienna, May 29-31, 2026. Written Thursday so the team can land Friday and execute.

## What is here

| File | When to use |
|---|---|
| `video-script.md` | Record the 60-second pitch video. Shoot Saturday night so it is ready by Sunday morning. |
| `slide-deck.md` | The 11-slide deck. Markdown for now; convert to Pitch or Google Slides on the flight. |
| `q-and-a-prep.md` | Read once Friday night, again Saturday night. The escalation matrix at the bottom is the only thing you need to memorize. |
| `track-reframes.md` | Open after Case Reveal Friday 6 PM. Pick the track and apply the slide swaps. |
| `demo-runbook.md` | Sunday morning two hours before the pitch. Run top to bottom. Do not improvise. |

## Order of operations

1. **Now (Thursday):** read all six files. Flag anything that is wrong about Apiary, the team, or the demo. Edit in place.
2. **Friday 6 PM (Case Reveal):** read briefs, pick track using `track-reframes.md`. Edit slides 1, 8, 11 to match.
3. **Saturday night:** record the 60-second video using `video-script.md`. Final slide pass.
4. **Sunday 11 AM:** run `demo-runbook.md` pre-flight checklist.
5. **Sunday 13:30:** pitch.

If something on stage goes wrong, the fallback paths in `demo-runbook.md` cover the most likely failures.
